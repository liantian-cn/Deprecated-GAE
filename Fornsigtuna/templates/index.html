一个简单的简繁转换工具，但不同于常见的转换工具针对字进行转换，这个工具针对詞。
比如大陆写“源代码”，台湾写“原始碼”。
又比如大陆写“餐台”，香港写“餐枱”，臺灣写“餐檯”。

转换规则来此维基百科ZhConversion.php

提供完整的api:

  • URL : /api
  • Method : POST
参数 是否必须 描述
text 是 待转换的的字符，UTF-8编码
html 否 默认返回一个json字符串，如果传入任意值的html参数，将返回一个html文本。
lang 是 目标翻译的语言，必须是下列之一：
  • zh-Hans - 通用简体(即只按字转换)
  • zh-cn - 大陆简体
  • zh-Hant - 通用繁体(即只按字转换)
  • zh-hk - 香港繁体
  • zh-tw - 台湾繁体
不区分大小写。
